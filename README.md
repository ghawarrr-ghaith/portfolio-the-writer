# Portfolio The Writer

A psychological thriller-inspired portfolio inspired by **Alan Wake** (*Remedy Universe*).

## Overview
"It's not a lake, it's an ocean." This project embodies the mysterious, clear, and dark atmosphere of Bright Falls. It features typewriter aesthetics, high contrast lighting mode, and a narrative-driven UI.

## Tech Stack
-   **Framework:** React 19 + TypeScript
-   **Build Tool:** Vite
-   **Styling:** Tailwind CSS v4
-   **Animation:** Framer Motion (optional, but CSS animations used here)
-   **Icons:** Lucide React

## Key Features
-   **Themed UI:** Narrative-driven design with typewriter fonts and stark contrast.
-   **Interactive Elements:** Vignette effects, smooth fade-ins, and light/dark toggle.
-   **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
-   **Flat Architecture:** Inlined component logic for a sleek, manageable codebase.

## Project Structure
```text
src/
├──├── data/         # Content and configuration
├── sections/     # Page sections (Hero, About, Projects, etc.)
└── App.tsx       # Main application entry
```

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone [repo-url]
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-the-writer
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
